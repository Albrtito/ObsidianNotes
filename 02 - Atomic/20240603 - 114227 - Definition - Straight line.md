# Straight line: 
An straight line can be written in several forms:

**Slope-Intercept form**:
$$
y = mx + d
$$
**General form:**
$$
ax + by + c = 0
$$
**Point-Slope:**
$$
y = y_0 + m(x-x_0)
$$
