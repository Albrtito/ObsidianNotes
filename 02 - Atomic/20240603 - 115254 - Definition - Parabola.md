All points with: 
+ **The same distance to a fixed point →** Focus
+ **The same distance to a fixed straight line** → Directrix

It is defined by: 
$$
\boxed{y = ax^2 + bx + c : a\not = 0}
$$

**Remarks:**
+ The definition for half a parabola is: 
$$
y = \sqrt{ax + b}
$$
