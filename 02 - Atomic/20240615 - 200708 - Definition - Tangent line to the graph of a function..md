# Tangent line to the graph of a function f: 

> [!NOTE] Definition:  
> Given the definition of derivative we define the tangent line to the graph of a function f(x) at the point (a, f(a)) as: 
>$$
y = f(a) + f'(a)(x-a)
>$$
