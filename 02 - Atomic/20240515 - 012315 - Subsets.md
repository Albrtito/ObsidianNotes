> [!NOTE] Subsets
> + Using the [[20240515 - 012600 - Definition - Binomial coefficients|Binomial coefficients]]
> + **The order does not matter**
> + " From n choose r": Choose pairs of r from a set of n elements
>

$$
\binom {n}{r} = {n!\over r!(n-r)!}
$$
