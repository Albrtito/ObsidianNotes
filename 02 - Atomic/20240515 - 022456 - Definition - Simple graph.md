# Simple graph:
A simple graph is a Pseudograph that has: 
+ **No loops**
+ **No multiedges**

They are widely studied in graph theory because this two properties simplify a lot of problems. Some of the well known simple graphs are:

+ [[Complete graph]]
+ [[Path graph]]
+ [[Cycle graph]]
+ [[Wheel graph]]
+ [[Complete bipartite graph]]
+ [[The n-cube graphs]]
